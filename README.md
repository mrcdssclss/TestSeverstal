This is a **test task for an internship at Severstal** as an **Intern Java Developer**.  

The main goal was to create a **basic backend application for a veterinary clinic**.

---

## Tech Stack

- Java 17  
- Spring Framework (Spring Boot, JPA/Hibernate)  
- PostgreSQL  

---

## Description

- Implements **CRUD operations** for managing:  
  - Patients  
  - Doctors  
  - Illnesses  
  - Appointments  
- Backend only, **no UI provided**.

---

## How to Run

1. Make sure **PostgreSQL** is installed and running.  
2. Update `application.properties` with your database credentials.  
3. Run the application: mvn spring-boot:run
4. Use Postman or curl to interact with the API.
